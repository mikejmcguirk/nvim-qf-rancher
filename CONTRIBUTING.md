## Contributing:

- Plugin development requires Neovim built with LuaJIT
- Testing is performed with nlua (https://github.com/mfussenegger/nlua) and Busted
- Formatting is performed with stylua (https://github.com/JohnnyMorganz/StyLua)
- Documentation is generated using vimcats (https://github.com/mrcjkb/vimcats)
